# Netflix Data Analytics Project 📊

## Overview 📌
This project focuses on analyzing Netflix content data to extract meaningful business insights. The workflow covers the complete data analytics pipeline, starting from raw data processing to interactive dashboard creation.

The project includes:
- Loading and processing dataset using Python
- Performing Exploratory Data Analysis (EDA)
- Cleaning and transforming data
- Running SQL queries in MySQL for business insights
- Building an interactive dashboard in Power BI

The goal of this project is to identify content trends, genre popularity, audience targeting, and regional content distribution to support data-driven decision-making.

---

## Dataset 🗂️
The dataset contains Netflix Movies and TV Shows information, including:

- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

**Dataset Size**
- Rows: 8800+
- Columns: 12

## Project Workflow ⚙️

### 1. Data Loading 📥
The dataset was imported into Python using Pandas for initial exploration.

### 2. Exploratory Data Analysis (EDA) 🔍
EDA was performed to understand content patterns, distributions, and trends.

### 3. Data Cleaning 🧹
Raw data was cleaned by handling missing values, removing duplicates, and transforming columns for analysis.

### 4. SQL Analysis Using MySQL 🗄️
SQL queries were used to extract business insights such as content growth, country distribution, and rating analysis.

### 5. Dashboard Development in Power BI 📈
An interactive dashboard was created to visualize key insights.

## Results and Insights 💡
Major findings from the analysis include:
- Movies dominate the Netflix catalog compared to TV Shows
- Major content growth occurred between 2018–2020
- TV-MA and TV-14 are the most common ratings
- Drama and International Movies are the most popular genres
- United States and India contribute the highest content volume

These insights help understand:
- Content strategy
- Audience targeting
- Regional market contribution
- Genre demand trends
